# All of Us Researcher Field Guide — Brand Assets v1.0

This package contains production-ready vector assets derived from the approved brand identity.

## Logo files
- `logo/logo-primary.svg` — stacked primary lockup
- `logo/logo-horizontal.svg` — preferred website/header lockup
- `logo/logo-mark.svg` — branch-point symbol
- `logo/logo-mark-white.svg` — reversed logo mark
- `logo/wordmark.svg` — wordmark without symbol
- `logo/favicon.svg` — scalable favicon source

## Icon files
Ten original SVG icons are included:
workflow, cohort, phenotype, analysis, validation, pipeline, workspace, data, reference, and help.

## Brand colors
- Primary Navy: `#183A5A`
- Secondary Blue: `#2F6FA5`
- Light Blue: `#EAF3FA`
- Accent Gold: `#C7A44D`
- Body Text: `#2B2B2B`

## Typography
The SVG wordmarks reference:
- Merriweather for headings/wordmark
- Inter for body/tagline

Fonts are referenced by family name and are not embedded.

## Usage
Use SVG as the source format. Export PNG derivatives only when a platform does not accept SVG.
