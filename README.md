# The All of Us Researcher Field Guide

A practical guide to reproducible EHR, genomic, and wearable data analysis in the All of Us Researcher Workbench.

## Local preview

1. Install Ruby and Bundler.
2. Run `bundle install`.
3. Run `bundle exec jekyll serve`.
4. Open `http://127.0.0.1:4000/aou-researcher-field-guide/`.

## Important

Replace every instance of `YOUR-USERNAME` in `_config.yml` before publishing.
