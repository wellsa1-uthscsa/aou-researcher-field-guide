# The All of Us Researcher Field Guide

Practical methods for reproducible EHR, genomic, and wearable data analysis in the All of Us Researcher Workbench.

{: .note }
This guide is designed around short, searchable pages. Each page answers one practical question and includes real code, expected outputs, validation steps, and common pitfalls.

## Start here

New to the Researcher Workbench? Begin with [How to Use This Guide]({{ site.baseurl }}{% link getting-started/how-to-use-this-guide.md %}), then continue to [Understanding the Workspace]({{ site.baseurl }}{% link getting-started/workspace-overview.md %}).

## What this guide covers

- Building and validating EHR cohorts
- Working with OMOP concepts and BigQuery
- Joining genomic and EHR data
- Using Hail and structural-variant resources
- Engineering Fitbit and sleep phenotypes
- Running reproducible association and sensitivity analyses
- Turning notebooks into inspectable, reusable pipelines

## Guiding principle

> Every analytical step should produce something you can inspect, validate, save, and explain.
